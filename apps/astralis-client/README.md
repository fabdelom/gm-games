# Astralis Client (v0.1 bootstrap)

Ce dossier hébergera le client web Angular + Phaser pour **Les Terres d’Astralis**.

## Scope v0.1
- Intégration Angular shell.
- Intégration Phaser dans un composant dédié.
- Connexion Nakama + room map.
- Affichage map isométrique `map_001`.
- Synchronisation de position multijoueur.

## Prochaines commandes prévues
- `yarn astralis:client:init` (scaffold Angular)
- `yarn astralis:client:serve`
